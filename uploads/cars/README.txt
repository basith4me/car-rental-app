CAR IMAGES UPLOAD DIRECTORY
============================

This directory stores uploaded car images.

IMPORTANT NOTES:
- Only image files (JPG, JPEG, PNG, GIF) should be here
- Maximum file size: 5MB
- Files are automatically named: car_[unique_id].[ext]
- Do NOT manually delete files unless you're sure they're not in use
- Check database before deleting any image file

SECURITY:
- PHP execution is blocked by .htaccess
- Directory listing is prevented by index.php
- Only upload images through the admin panel

For more information, see IMAGE_UPLOAD_GUIDE.md in the project root.
