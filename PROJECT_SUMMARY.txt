OVERVIEW
--------
A complete, fully functional car rental management system built with
plain HTML, CSS, Bootstrap, JavaScript, PHP, and MySQL.

FEATURES IMPLEMENTED
-------------------
✓ User Authentication (Admin & Customer)
✓ Admin Dashboard with Statistics
✓ Add/Edit/Delete Cars
✓ View All Bookings
✓ Manage Users (View/Delete)
