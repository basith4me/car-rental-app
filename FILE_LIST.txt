DATABASE SCHEMA DETAILS

USERS TABLE
-----------
Fields:
- id (Primary Key)
- username (Unique)
- email (Unique)
- password (Hashed)
- full_name
- phone
- address
- role (admin/customer)
- created_at

CARS TABLE
----------
Fields:
- id (Primary Key)
- car_name
- brand
- model
- year
- color
- registration_number (Unique)
- seat_capacity
- price_per_day
- fuel_type
- transmission
- image
- status (available/booked/maintenance)
- description
- created_at

BOOKINGS TABLE
--------------
Fields:
- id (Primary Key)
- user_id (Foreign Key)
- car_id (Foreign Key)
- booking_date
- start_date
- end_date
- total_days
- total_price
- status (pending/confirmed/cancelled/completed)
- payment_status
